# Privacy Policy (Draft)

This service processes Gmail data only for user-authorized automation:

- Classifying job-related emails
- Applying user-visible labels
- Creating drafts when enabled

## Data We Store

- Account profile data (email, preferences)
- Encrypted OAuth refresh tokens
- Processing metadata (run status, logs, audit events)

## Data We Do Not Sell

No user data is sold to third parties.

## User Controls

- Disconnect Gmail at any time
- Pause/resume automation
- Request account/data deletion

## Retention

Operational metadata is retained for service reliability and abuse prevention.
Deletion requests disable accounts and remove associated tenant records.
